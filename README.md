# Calcify

A modern full-stack MERN Calculator Web App with a clean UI, responsive design, real-time calculations, and MongoDB-powered calculation history.

---

## ✨ Features

- Modern and aesthetic calculator UI
- Responsive design for mobile and desktop
- Real-time calculations
- Save calculation history in MongoDB
- Delete history feature
- Built using MERN Stack
- Smooth hover animations
- Beginner-friendly React code

---

# 🛠️ Tech Stack

## Frontend
- React.js
- CSS3
- Axios

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

# 🚀 Installation

## 1. Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

---

## 2. Open Project Folder

```bash
cd Calculator-Web-App
```

---

# ⚙️ Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file inside backend folder:

```env
MONGO_URL=your_mongodb_connection_string
```

Start backend server:

```bash
npm start
```

Backend runs on:

```bash
http://localhost:5000
```

---

# 💻 Frontend Setup

Open a new terminal:

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```
